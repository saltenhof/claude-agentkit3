# Integration Stabilization

Dieser Kontext formalisiert den engen Ausfuehrungsvertrag fuer spaete
systemische Integrations- und Stabilisierungslagen innerhalb von
`story_type=implementation`.

Enthalten sind:

- Manifest und Budget als kanonische Steuerungsobjekte
- Zustandsmaschine der Stabilisierungsschleife
- offizielle Kommandos und Telemetrie
- Invarianten gegen Freibrief- oder Guard-Lockerungslogik
